USAGE						*ogpmd-usage*

This plugin provides commands for basic interaction. See |ogpmd-commands|.

==============================================================================
COMMANDS					*ogpmd-commands*

*:Ogpmd* {url}
	Fetches the HTML title and OGP image (if available) from the specified
	{url}. Inserts the title as a Markdown link (`[title](url)`) and the
	OGP image's Base64 encoded string directly after the current cursor line.
	Example:
>
	:Ogpmd https://github.com
<
	(`[GitHub: Let’s build from here · GitHub](https://github.com)` and
	 the raw Base64 string of the OGP image might be inserted below the current line)
